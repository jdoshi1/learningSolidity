### Contract linter:

* [Remix browser](http://remix.ethereum.org/#version=soljson-v0.4.18+commit.9cf6e910.js)

* [Solhint](https://github.com/protofire/solhint)

    Run solhint on the terminal as:
    ```bash
    solhint MyFirstContract.sol
    ```
